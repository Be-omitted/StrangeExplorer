# StrangeExplorer

StrangeExplorer是一个也许略显奇怪的本地资源管理器，支持多种类型文件的本地储存与标签管理，同时提供基本爬虫与相关接口以进行快捷的下载导入。

## 功能
### 独特的Tag与Tag Group机制

在某些网站中，Tag往往零散且混乱，代指来自某一个游戏可能有数种名称（如"东方", "东方Project"...），这为搜索提供了些许问题。

而本软件提供将多个Tag归类为同一个Tag Group的功能，同时允许一个Tag归类于多个Tag Group（参考爱丽丝，爱丽丝和爱丽丝），并编写注释以供查询。

并对Tag Group添加if条件，必须包含指定Tag才能添加该Tag Group，以防某些添加作品角色Tag却没有加入作品Tag的情况。

同时通过特殊的检索功能以提供仅需搜索Tag也可同时找到其父Tag Group的功能。

这所需要的基础Tag库请从Releases获取，这包含了常见的Tag。


### 便于分享的管理与分离机制
所有此软件创建的文件库都是可以异地使用的，同时，你可以直接操作其内部文件。

同时，提供副本制作功能以将其大文件库分离出指定文件与Tag。

这为私下交流与传播提供了帮助。

## 简单教程
### 开始
选择空文件夹以在其中创建文件库（推荐使用Releases的标准库）。
### 信息栏
在窗口最下方的文字信息栏会提供帮助或返回错误信息。
### 导入爬虫
选择下载->导入源->选中爬虫文件夹（须解压缩）
完成导入
### 下载
在下载窗口选中爬虫，输入关键字~~如jm号~~以开始下载

切换 管理->查询位置->暂存区，点击选择以挑选下载文件夹
